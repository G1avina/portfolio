Pac man Game

This project is a very simple pacman game made using html, css and javascript. In this game 'x' is your character, '@' are fruits you need to collect, '.' are empty slots and 'c' are the ghost you need to avoid

Yatzy Game

This project is a Yatzy game made using html, css and javascript. In this game you must try to get the highest score possible by rolling dices.

Yatzy Game with Server

This project is a Yatzy game made using html, css and javascript and a PHP server. This project is similar to the yatzy project except all calculation are performed in a php localHost server. This version of the game also includes a leaderboard



Emergency Waitlist Website

This project is a website using html, css, javascript, a PHP server and a postgres database. This project simulates a hospital queue for their patients. The admins can register new admins and patients or view and remove patient from the patient queue Patients can login and view their profile and place in the queue.
